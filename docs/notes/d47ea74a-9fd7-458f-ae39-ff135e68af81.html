API

My personal api is available at api.muensterer.xyz (redirects to next.muensterer.xyz/api) The available routes are: | Route | Description | | :------------- | :------------- | | /dennis | Returns information about me | | /events | A list of events in my life (used for muensterer.xyz/life) | | /social | Returns a json of my social-urls | | /social/ + (facebook, twitter, instagram, linkedin, github, youtube, spotify, telegram) | Gets a redirect to the given profile page | | /playlist | Returns a json of my spotify playlists | | /playlist/ + (current) | Gets a redirect to a specific playlist |

Much of the data made available through this api is stored in gists.
